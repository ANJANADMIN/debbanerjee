By the end of this project I want to be able to use HTML5, CSS3 and Bootstrap to make any webpage or app layout that I want.

Why am I learning it?

I want to learn to create beautiful webpages and app interfaces.

It is not something that will help me get a raise or anything. I just love to learn how to make things. In this project I want to learn to make beautiful interactive webpages.

This project is the first step towards my end goal of building an app/web based service that will help people to solve some real problem in their life.
